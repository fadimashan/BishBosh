Bish-Bosh

Bish-Bosh lists all numbers between 1 and "n", but some exceptions: If the number is evenly divisible with 3 listed ‘Bish’ instead of the number. If the number is evenly divisible by 4, ‘Bosh’ is listed instead the number. If the number is evenly divisible by both 3 and 4, write "Bish-Bosh" instead of the number.
